# 💼 Indira HK — Personal Portfolio

Personal portfolio website for **Indira HK**, Full Stack Developer based in Bengaluru, India.

## 🌐 Live Site
> https://indirahk.github.io/portfolio *(update after deploying)*

## 🚀 Tech Stack
Built with pure **HTML5, CSS3, and JavaScript** — no frameworks, no build tools.

## 📁 Project Structure
```
portfolio/
├── index.html    → Page structure & content
├── style.css     → All styling & animations
├── script.js     → Cursor, scroll reveal, active nav
├── .gitignore    → Git ignore rules
└── README.md     → This file
```

## 💡 Features
- Custom animated cursor
- Smooth scroll-reveal animations
- Responsive layout (mobile + desktop)
- Active nav highlighting on scroll
- Sections: Hero · About · Skills · Projects · Contact

## 🛠️ How to Run Locally
Just open `index.html` in any browser — no install needed!
